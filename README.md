#  RewindCodeFilm

Una página web básica para explorar y descubrir películas. Este proyecto muestra información de películas populares, incluyendo su título, sinopsis y calificación. Ideal como práctica de desarrollo web y diseño responsivo.

---


##  Introducción
es un proyecto web básico que permite a los usuarios navegar por una lista de películas. Se enfoca en la presentación y estructura del contenido, utilizando tecnologías web modernas y un diseño atractivo.

---

##  Características
- Muestra una lista de películas con imagen, título y descripción.  
- Diseño responsivo compatible con dispositivos móviles.  
- Buscador de películas.  
- Página de detalles para cada película .  

---

##  Tecnologías Utilizadas
- **HTML**  
- **CSS**  
- **JavaScript**  
- **PHP**
